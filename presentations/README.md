# Presentations

Training materials for use by Misheck and Justice during site visits.

| Folder | Used in programme | Audience | Format |
|--------|-------------------|----------|--------|
| `nextcloud-user-registration/` | Week 1 | Teacher (admin role) | PDF + Markdown |
| `nextcloud-classroom-use/` | Weeks 2–5 | Teachers | PDF + Markdown |

## nextcloud-user-registration
**"Registering Users and Organizing Classes"**
Covers: accessing the Accounts dashboard, creating class groups, registering student accounts, mobile access setup.
Use this in Week 1 to walk a teacher through setting up their class in Nextcloud.

## nextcloud-classroom-use
**"Managing Your Digital Classroom with Nextcloud"**
Covers: distributing read-only materials to a class, collecting homework via File Drop, collaborative live documents, using Talk for classroom communication.
Use this across Weeks 2–5 as each feature is introduced.

---
PDFs can be printed or viewed offline on the IDEA system.
Markdown files with images can be served from Nextcloud or Kolibri if configured.
