
## IDEA System — How Apps Are Accessed (from Koen, 2026-03-26)

Teachers dock an app disk to make an app available; undocking removes it.

Users (teachers and students) access apps via a web page served by the engine:
- Default URL: **`engine-1.local`** (opened in Chromium browser)
- This page lists all apps available on the network from all engines
- Each app shows its status: docked, undocked, starting, stopped, running, error
- When an app is **running**, its name becomes a clickable link — user clicks it to open the app

**Implication for documentation:** Users do NOT navigate directly to Kolibri/Nextcloud/Wikipedia URLs. They always start at `engine-1.local` and click through to the app from the status page.
