# Nextcloud — Quick Reference Card
_Keep this card. Use it every week._

---

## Log In

1. Open the browser → go to **`[NEXTCLOUD URL]`**
2. Enter your **username** and **password**

---

## Share a Folder with Your Class

1. Click **Files** in the top menu
2. Right-click your class folder → click **Sharing**
3. Search for your class group name (e.g. `Grade 5A`) → click it
4. Set permission to **View only** (students can read but not delete)
5. Click **Save**

---

## Collect Homework (File Drop)

1. Right-click a folder → **Sharing** → enable **File drop**
2. Students can upload files but cannot see each other's submissions
3. You see all uploaded files when you open the folder

---

## Check Who Has Been Active (Activity Log)

1. Click your **name** (top right) → **Activity**
2. You see a log of all logins, file views, and uploads
3. Check this at every visit — look for teacher and student logins

---

## Create a New Student Account

1. Click your **name** (top right) → **Accounts**
2. Click **New user** → enter name, username, password
3. Add the student to their class group under **Groups**

---

## Common Problems

| Problem | What to do |
|---------|-----------|
| Student can't see the shared folder | Check they are in the class group: Accounts → Groups |
| Student uploaded to wrong folder | Remind them to use the Drop Zone link shared in Talk |
| Activity log is empty | Check the Activity app is enabled — ask Tapiwa |
